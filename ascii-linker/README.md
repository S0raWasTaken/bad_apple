# ascii_linker

Embed the ASCII videos generated by [asciic](../asciic) into your Rust binary.

## Usage

```rust
use ascii_linker::link_bapple;

const FRAMES: &[&str] = link_bapple!("./animation.bapple");

fn main() {
    for frame in FRAMES {
        print!("\x1B[2J\x1B[H{}", frame);
        std::thread::sleep(std::time::Duration::from_millis(33));
    }
}
```

## What's a `.bapple`?

A tar archive of zstd-compressed ASCII frames, automatically generated by [asciic](../asciic).<br>
The macro decompresses everything at compile time and generates a static array.

## Why?

- Zero runtime overhead
- Single binary distribution
- No file I/O needed
- Works in bare-metal environments (kernels, bootloaders)

## Trade-off

Larger binary size, but instant frame access.
